# 💻 Desafío 3 - Funciones JavaScript

Este proyecto corresponde al **Desafío 3 - Funciones**, donde se practican funciones de expresión, arrow functions, eventos del DOM y cambios de estilos usando JavaScript.

---

## 🚀 Vista del proyecto

El desafío incluye ejercicios separados para trabajar:

- Transformación de funciones tradicionales.
- Uso de arrow functions.
- Eventos `click`.
- Cambio de colores desde JavaScript.
- Eventos de teclado con `keydown`.
- Creación dinámica de elementos HTML.

---

## 🧩 Ejercicios realizados

### 1. Función de expresión

Archivo:

```text
1_funcion.js
```

Se transforma una declaración de función en una función de expresión.

### 2. Arrow function

Archivo:

```text
2_arrow.js
```

Se transforma una función tradicional en una arrow function de una línea.

### 3. Pintar elemento

Archivos:

```text
pintar.html
assets/js/script.js
```

Al hacer clic sobre el elemento `hello`, este cambia su color de fondo a amarillo.

### 4. Colores y teclado

Archivo:

```text
4_colores.html
```

Incluye cuatro divs de colores que cambian a negro al hacer clic. También permite cambiar el color del div `key` usando las teclas `a`, `s` y `d`, además de crear nuevos divs con las teclas `q`, `w` y `e`.
